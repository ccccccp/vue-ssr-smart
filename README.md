# vue-ssr

## 支持的功能

1.公共模块分离
2.路由懒加载
3.服务端渲染数据预取
4.本地调试热模块替换

## 需要支持的浏览器

1. MS IE11(无需支持 IE10 及以下版本)
2. MS Edge
3. Google Chrome
4. Mozilla Firefox
5. Apple Safari 10+

## 启动项目


```bash
#安装依赖
npm install

# 本地需要全局安装cross-env包 
npm install -g cross-env

# 启动本地调试模式
npm run dev

# 构建并运行服务
npm run start

```
